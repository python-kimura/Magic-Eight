# Magic 8 Ball

This is a simple program that simulates the operation of the classic toy "Magic Eight Ball". It provides random answers to your questions, adding an element of uncertainty and fun to the decision-making process.

Main functions:

Generates random answers from a predefined list.

Supports reuse for new questions.

Easily expandable list of possible answers.


# Магический Шар Восьмерка

Это простая программа, имитирующая работу классической игрушки "Магический Шар Восьмерки". Она предоставляет случайные ответы на ваши вопросы, добавляя элемент неопределенности и веселья в процесс принятия решений.

Основные функции:

Генерирует случайные ответы из предустановленного списка.

Поддерживает повторное использование для новых вопросов.

Легко расширяемый список возможных ответов.
